# Shared Agent Skills

These are the canonical, cross-agent skills for this dotfiles setup.

- Format: Agent Skills-compatible `SKILL.md` directories
- Primary install target: `~/.agents/skills`
- Pi consumes these directly
- Other agents can consume them directly or via lightweight wrappers

Current skills:

- `git-safety`
- `tdd`
- `dotfiles-change`
- `nix-change`
- `debug-repro`
- `code-review`
- `commit-prep`
