# Dotfiles

This repo contains my dotfiles. Feel free to use them as you wish. They are pretty simple and you'll probably want to customize them to your needs.

## Installation

These dotfiles are manged with [chezmoi](https://www.chezmoi.io). To install them, run:

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply ThePaulMcBride
```
